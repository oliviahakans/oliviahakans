SPRINTS

The project was carried out over a period of four weeks, divided into four sprints with their own focus points.

SPRINT 1

  • Understand the provided code from EPB
  • Select files from the file system and make sure that they can be applied to the chat for questioning
  • Implement an index for the file.
  • Create a website for the project display

SPRINT 2

  • Created a login feature
  • Worked on splitting files to limit tokens, bounded to manual handling only for this sprint
  • Saving chat history in the RavenDB database connected to the users
  • FIGMA prototype implemented

SPRINT 3

  • Implement the FIGMA design
  • Continued woriking on splitting files automatically by parapraphs
  • Bug solved saving chats

SPRINT 4

  • Finalised the prodject display website
  • Bug solved the entire code
  • Implemented the last figma features
  • Planned for the expo
